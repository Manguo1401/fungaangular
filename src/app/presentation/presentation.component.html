Aaron Hartnell, développeur web full-stack

Description

Bonjour, mon nom est Aaron Hartnell. Je suis un auto-entrepreneur franco-anglais spécialisé et passioné dans le développement web full-stack. Mon cursus professionnel et universitaire avec un master en géomatique m'ont apporté des clés solides pour répondre aux problématiques que peut avoir un site web à fonctionnalités complexes. La performance et la structure du code et de la base de données sont mes priorités lorsque je développe afin d'obtenir une performance optimale lors de la mise en ligne de votre site web et de la vitesse de son chargement.

Avide de connaitre les dernières avancées sur le web, je cherche toujours les nouvelles technologies pour les sites web de demain. Je priviligie toujours l'écoute du client et me concentre à apporter la meilleure solution en fonction de son budget et objectif. Etant bilingue anglais/français je peux aussi traduire votre site dans la langue de Shakespeare.

Je suis actuellement basé dans le Golfe du Morbihan mais je n'hésite pas à me déplacer si c'est nécessaire.

Langages et librairies utilisés

  • Angular 2/6, Node.js, Express
  • Symfony 3
  • OpenLayers 4, Leaflet
  • D3.js, Charts.js
  • JQuery
  • Bootstrap 4, CSS GRID
  • MySQL, PostreSQL
  • MongoDB
  • ...
